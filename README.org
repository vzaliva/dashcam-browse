* Install
** Depedencies  
   - Python3
   - [[http://bottlepy.org/docs/dev/][Bottle]]
   - [[https://requests.readthedocs.io/en/master/][Requests]]
   - [[https://github.com/martinblech/xmltodict][xmltodict]]
    
  Using *pip*:

     =pip3 install bottle requests xmltodict=

* Run
  =python3 dashcam-browse.py=
